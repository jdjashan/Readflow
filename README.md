We Present **ReadFlow**

An Adaptive and usefull tool geared towards helping students who have leaning disabillities and/or trouble reading. 
Or even elders who want to stay engaged whilst reading!

With the Use of ElevenLabs AI and Gemini, you are able to create or upload books. Then the AI will read it out at different speeds and affection/ emotion levels.
The user is analyzed through Presage AI, which can sense their emotions and change the reading abits through this.

Frontend created using React (HTML, and JSX)

It s very tranformational in the audibook and game and we feel could make a big difference.

Hope you enjoy the project!
